Project Hello Quad

3D Graphics

Project Overview

This project was an assignment focused on implementing shaders on a simple quad using WebGL. The main objective was to apply object transformations based on mouse movement and to render a smooth rainbow gradient color that changes according to the size of the model.

Tasks Completed

  • Vertex Shader: Implemented rotation logic by applying a transformation matrix that responds to mouse input.
  • Fragment Shader: Created a dynamic color effect that changes gradually based on the modelâ€™s scale, using HSV to RGB conversion to generate a rainbow gradient.
  • OpenGL Class Implementations: Designed and implemented modular classes to encapsulate OpenGL resources such as textures, vertex buffers, index buffers, and vertex arrays to promote code reusability and clarity.

Reflection

One of the most challenging parts of this project was setting up the web debug environment. Configuring CMake and managing environment variables proved to be quite difficult, but overcoming these challenges helped me gain a deeper understanding of build systems and project setup for WebGL development.
