Project Procedural Geometric Modeling

3D Graphics

Project Overview

This project focused on generating 3D geometric models procedurally using OpenGL and GLSL. The primary goal was to implement various shape creation functions (such as cube, sphere, cylinder, cone, torus) using parametric equations, while constructing index buffers and vertex attributes for rendering.

Tasks Completed

  • Procedural Mesh Generation: Implemented geometry generation logic for planes, cubes, spheres, cylinders, cones, and tori, using parametric formulas and trigonometric functions.
  • Vertex Layout Definition: Defined custom vertex structures with position, normal, and UV attributes and mapped them into OpenGL buffer layouts using attribute descriptors.
  • Caps and Topology Logic: Added logic to construct top and bottom caps for cylinders and cones to ensure watertight geometry.

Reflection

A key learning experience was handling mesh topology and index buffer generation. Carefully managing winding order and understanding how indices map to vertices was crucial. Writing reusable and modular geometry functions also deepened my understanding of mesh data flow in a rendering pipeline.
