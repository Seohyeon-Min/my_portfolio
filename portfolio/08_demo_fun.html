Project Fly Simulation

3D Graphics

Project Overview

Created a flight simulator with procedural clouds and visual effects.

Press the spacebar to move forward.

You can change direction using the WASD keys.

Tasks Completed

  • Implemented toon shading for stylized rendering
  • Generated clouds using 3D noise techniques
  • Applied atmospheric fog
  • Added post-processing vignette effect
  • Procedurally placed clouds that generate as the player moves forward
