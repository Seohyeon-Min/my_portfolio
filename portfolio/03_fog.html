Project Fog

3D Graphics

Project Overview

This project focused on implementing vertex and fragment shaders in GLSL (OpenGL ES 3.0) to apply exponential fog effects to 3D models. The goal was to deepen my understanding of the graphics pipeline and enhance realism by simulating atmospheric depth in 3D scenes.

Tasks Completed

  • Wrote vertex and fragment shaders using GLSL, targeting OpenGL ES 3.0.
  • Implemented exponential fog using the formula: 1 - exp2(-density² × distance² × log2).
  • Properly transformed model space coordinates to view space by applying model, view, and projection matrices.
  • Debugged and fixed incorrect fog results by identifying that the model matrix was missing in fog coordinate computation.
  • Used sampler2D for texture mapping and ensured alpha blending with fog color.

Reflection

This project helped solidify my understanding of how 3D transformations work in the graphics pipeline, especially the importance of applying the model matrix when converting positions to view space. I initially struggled with incorrect fog visuals until I realized that only the view matrix was being applied. This experience taught me the importance of precise space conversions in shader programming and gave me hands-on practice implementing both smoothstep and exponential fog models.
